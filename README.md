# VedLab - A Figma Clone

VedLab is a powerful design tool, built to replicate the features and functionalities of Figma. It allows designers and teams to collaborate seamlessly on vector-based designs in real-time. With an intuitive interface and robust features, VedLab aims to be a reliable solution for creative professionals and design teams.

## Features

- **Real-time Collaboration**: Work on designs simultaneously with teammates.
- **Vector-based Design**: Create scalable vector graphics with precision.
- **Interactive Prototypes**: Build prototypes and animate interactions for realistic user experiences.
- **Version Control**: Keep track of design changes with automatic version history.
- **Cloud Integration**: Save and access your work from anywhere via cloud storage.

## Installation

To get started with VedLab locally, follow these steps:

### Prerequisites
Ensure you have the following installed on your system:
- Node.js (v14 or higher)
- npm (v6 or higher)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/VEDANTDHAVAN/VEDLAB.git
   cd vedlab
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. Open your browser and go to `http://localhost:3000` to start using VedLab locally.
